# hunainkaisar.com

This repository contains the source code and assets for my personal website, **https://hunainkaisar.com**. The website is deployed using **Netlify** and is publicly accessible over HTTPS.

## Table of Contents
- About
- Repository Structure
- Usage
- Contributing
- License

## About
This website serves as my personal portfolio, showcasing my background, skills, projects, and contact information. The site is automatically built and deployed from this GitHub repository using Netlify.

## Repository Structure
- `index.html` – Main HTML entry point
- `assets/` – Static assets such as images, styles, and built resources
- `package.json` – Project dependencies and scripts
- Configuration files – Tooling and build configuration used for deployment

## Usage
To view the website, visit:

**https://hunainkaisar.com**

Any changes pushed to the repository are automatically built and deployed by Netlify.
